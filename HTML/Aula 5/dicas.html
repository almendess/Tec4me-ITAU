Dicas Para Aprender HTML

Ao acessar uma página web através de um navegador, ele é capaz de interpretar o código HTML e renderizá-lo de forma compreensível para o usuário final. Com as configurações definidas por meio de diversas tags que essa lingugagem dispõe, é possível exibir textos, botões, formulários etc.

Atualmente a HTML encontra-se na versão 5 e é padronizada pelo W3C (World Wide Web Consortium), uma organização internacional responsável por estabelecer padrões para a internet, como a linguagem XML, CSS e o SOAP.

Estrutura básica de uma página HTML

Na estrutura básica de uma página HTML, podemos ver as principais tags que são necessárias para que o documento seja corretamente interpretado pelos browsers.

<!DOCTYPE html>
<html>
<head>
<meta charset="UTF-8"/>
<title>Documento</title>
</head>
<body>
Todo o conteúdo da página vai dentro das tags de abertura e fechamento de body.
</body>
</html>

DOCTYPE: a instrução DOCTYPE deve ser sempre a primeira a aparecer em uma página HTML para indicar ao browser qual versão da linguagem usada. Nesse caso, estamos trabalhando em HTML5, versão na qual a declaração do DOCTYPE é bastante simples, como podemos ver no exemplo;

HTML: a abertura e fechamento da tag html, serve para delimitar o documento.Sendo assim, todas as demais tags da página devem estar nesse espaço;

HEAD: a abertura e fechamento da tag head, serve para definir o cabeçalho do documento. O conteúdo nesse espaço não é visível no browser, mas contém instruções sobre seu conteúdo e comportamento. Dentro dessa tag, por exemplo, podem ser inseridas folhas de estilo e scripts;

META: a tag meta, nesse caso, especifica qual conjunto de caracteres(character set ou charset) será usado para renderizar o texto da página. O UTF-8 contém todos os caracteres dos padrões Unicode e ASCII, sendo, portanto, o mais utilizado em páginas web. A mesma tag meta, porém com outros atributos, pode ser utilizada para outros fins, como na SEO (Search Engine Optimization);

TITLE: a tag title define o título da página, aquele que aparece na janela/aba do navegador;

BODY: a abertura e fechamento da tag body, serve para marcar o espaço no qual deve estar contido o conteúdo visual da página. As demais tags que representam o texto, botões e outros, devem ser adicionados nesse intervalo.


Cabeçalhos do HTML

Cabeçalhos são normalmente utilizados para identificar páginas e seções e possuem aparência diferenciada do restante do texto. No HTML há seis níveis de cabeçalhos/títulos que podem ser utilizados por meio das tags h1, h2, h3, h4, h5 e h6, sendo o h1 o maior/mais relevante e h6 o menor/menos relevante.

Importante: De acordo com as regras de SEO, é recomendado que uma página possua apenas uma tag h1 que indique seu assunto, pois essa tag informa aos motores de busca qual sua principal palavra-chave.

<h1>Título de Nível 1</h1>
<h2>Título de Nível 2</h2>
<h3>Título de Nível 3</h3>
<h4>Título de Nível 4</h4>
<h5>Título de Nível 5</h5>
<h6>Título de Nível 6</h6>

Listas no HTML

Listas são elementos úteis para organizar e ordenar itens que estão relacionados de alguma forma. No HTML é possível criar ate três tipos de listas: Ordenadas (com a tag ol), Não Ordenadas (com a tag ul) e de Definição (por meio da tag dl).

Lista Não Ordenada:
<ul>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ul>
  • Item 1
  • Item 2
  • Item 3
Lista Ordenada:

Importante: No caso das listas ordenadas temos o atributo type que permite especificar o tipo de marcador a ser usado: números (padrão), números romanos ou letras.

Os valores permitidos para esse atributo são:

  • 1 - números (padrão)
  • i - números romanos minúsculos
  • I - números romanos maiúsculos
  • a - letras minúsculas
  • A - letras maiúsculas

<ol>
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3
<ol type="A">
<li>Item 1</li>
<li>Item 2</li>
<li>Item 3</li>
</ol>
  1. Item 1
  2. Item 2
  3. Item 3
Lista de Definição:

Importante: As listas de definição têm um comportamento um pouco diferente, uma vez que cada item é composto por um título(dt) e uma definição (dd), semelhante ao que ocorre em dicionários, nos quais temos os verbetes e suas definições.

<dl>
<dt>Título 1</dt>
<dd>Definição 1</dd>
<dt>Título 2</dt>
<dd>Definição 2</dd>
<dt>Título 3</dt>
<dd>Definição 3</dd>
</dl>
Título 1
Definição 1
Título 2
Definição 2

Definição 3
